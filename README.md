# Neon Aura AR ✧

> Transform your surroundings into a high-octane digital sanctuary.

**Neon Aura AR** is an immersive, hand-tracked augmented reality experience. It blends real-time computer vision with premium aesthetic shaders to turn your gestures into digital art.

### ⚡ Core Experience
*   **Precision Hand Tracking**: Powered by MediaPipe for sub-millisecond response times.
*   **Dynamic Soundscapes**: Real-time synthesized audio that reacts to your palm distance.
*   **Aesthetic Themes**: Cycle through *Cyberpunk*, *Galaxy*, *Lava*, and *Ocean*.
*   **Reactive Shaders**: Matrix-style rain and neon-bloom effects that follow your motion.

### 🧪 Technical Stack
*   **Vision**: MediaPipe Hands
*   **Graphics**: Canvas API with Screen-Blend Bloom
*   **Audio**: Web Audio API (Oscillator-based feedback)
*   **UI**: Glassmorphic CSS Engine
